# Full Implementation Workflow

## PHASE 1: RESEARCH & INPUT (Day 1)
[ ] Complete product_input_template.md in full
[ ] Gather 10+ real customer reviews — mine for exact language they use
[ ] Identify top 3 competitor ads currently running (use Meta Ad Library)
[ ] Define primary platform and secondary platform
[ ] Set campaign KPI targets (CTR goal, CPA target, ROAS target)

## PHASE 2: HOOK GENERATION (Day 1–2)
[ ] Run 02_hook_generator.md prompt with filled template
[ ] Generate 15 hooks
[ ] Internal review: score each hook 1–10 on: Relatability / Curiosity / Clarity
[ ] Select top 5 hooks for testing
[ ] Optional: test hooks as static image captions before filming

## PHASE 3: SCRIPT BUILDING (Day 2–3)
[ ] Run 03_problem_solution_script.md with top hook inserted
[ ] Generate 3 script variations (use 06_multi_hook_variations.md)
[ ] Run 07_conversion_psychology.md on your best script
[ ] Review: Does it sound like a real person talking? Read aloud.
[ ] Final script should be under 160 words for 60s

## PHASE 4: PLATFORM ADAPTATION (Day 3)
[ ] Run 04_platform_adaptation.md on winning script
[ ] Customize tone per platform (see notes in prompt)
[ ] Write captions for each platform
[ ] Prepare hashtag sets (use Section 5 from 08_complete_content_pack.md)

## PHASE 5: BRAND VOICE REVIEW (Day 3–4)
[ ] Run 05_brand_voice_customization.md if needed
[ ] Ensure language matches your brand positioning
[ ] Remove any phrases that feel "AI-generated" or unnatural
[ ] Have a real person read it aloud — does it feel human?

## PHASE 6: PRODUCTION (Day 4–7)
[ ] Brief your creator with the production notes from your chosen script
[ ] Shoot minimum 3 hook variations for the same script
[ ] Capture b-roll: product close-ups, before/after, usage shots
[ ] Raw footage review: check first 3 seconds — is it a thumb-stopper?
[ ] Edit: vertical 9:16, captions on, natural pacing

## PHASE 7: LAUNCH & TEST (Week 2)
[ ] Launch with $50–100/day test budget
[ ] Run 3 ad variations simultaneously (different hooks, same script body)
[ ] Monitor at 48hr: pause any ad with <25% 3-second retention
[ ] At 7 days: declare winning hook, scale budget on winner

## PHASE 8: OPTIMIZE (Week 3+)
[ ] Low 3s retention → New hook, same script
[ ] Low CTR → New CTA, same hook
[ ] Low conversion → Improve landing page or offer (not the ad)
[ ] High CPA → Narrow audience targeting
[ ] Scale winner: increase budget 20% every 48hrs
