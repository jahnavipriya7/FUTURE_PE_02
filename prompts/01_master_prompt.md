# 🎯 MASTER PROMPT: All-in-One UGC Ad Generator

> **Use this single prompt to generate a complete UGC ad script for any product.**

---

## HOW TO USE

Copy everything below the divider, paste it into ChatGPT / Claude / Gemini, and fill in the `[BRACKETS]`.

---

## THE PROMPT

```
You are an expert UGC (User-Generated Content) ad scriptwriter and performance marketer with 10+ years of experience creating viral content for TikTok, Instagram Reels, and YouTube Shorts.

I need you to create a complete, high-converting UGC ad script for my product.

--- PRODUCT DETAILS ---
Product Name: [YOUR PRODUCT NAME]
Product Category: [e.g., skincare / SaaS / fitness / food]
Key Benefit #1: [MAIN BENEFIT]
Key Benefit #2: [SECONDARY BENEFIT]
Key Benefit #3: [THIRD BENEFIT]
Target Audience: [WHO IS THIS FOR — age, gender, pain point]
Primary Pain Point: [WHAT PROBLEM DOES THIS SOLVE?]
Price Point: [PRICE]
Offer / CTA: [e.g., "50% off for the next 24 hours" / "Free trial"]
Platform: [TikTok / Instagram Reels / YouTube Shorts / Facebook]
Tone: [e.g., relatable, energetic, emotional, educational, humorous]

--- YOUR TASK ---
Generate the following:

1. **5 Hook Variations** (first 3 seconds) — use pattern interrupts, curiosity gaps, bold statements, and relatable scenarios

2. **Full 30-second Script** using the PAS framework:
   - Problem (0–5s): Agitate the pain point
   - Agitate (5–15s): Make them feel the frustration
   - Solution (15–25s): Introduce the product as the hero
   - CTA (25–30s): Clear, urgent call-to-action

3. **On-Screen Text Suggestions** — 3–5 text overlays that reinforce the message

4. **Caption** (150 words max) with relevant hashtags

5. **A/B Test Suggestion** — which element to test first and why

Format the script with [VISUAL], [VOICEOVER], and [TEXT OVERLAY] labels for easy production.
```

---

## OUTPUT EXAMPLE STRUCTURE

```
HOOK VARIATIONS:
1. "I can't believe I used to waste $200/month on..."
2. "POV: You finally found the thing that actually works"
3. "Stop scrolling if you have [pain point]"
4. "This product saved my [outcome]"
5. "No one talks about how [relatable problem]..."

FULL SCRIPT:
[VISUAL] Creator looking frustrated at [problem scenario]
[VOICEOVER] "I was so tired of [pain point]..."
[TEXT OVERLAY] "Sound familiar? 👇"
...
```

---

## TIPS

- **Hook is everything.** The first 1.5 seconds determines if someone watches or scrolls.
- **Be specific.** Replace generic language with exact numbers and outcomes.
- **Sound optional.** 85% of social media videos are watched on mute — make text carry the message.
