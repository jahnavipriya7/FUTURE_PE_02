# 📱 PLATFORM ADAPTATION — TikTok, Instagram, YouTube, Facebook, Pinterest

> **Same product, different platform rules. This prompt adapts your script for each.**

---

## THE PROMPT

```
You are a social media platform specialist. I have a UGC ad script that needs to be adapted for multiple platforms.

--- BASE SCRIPT ---
[PASTE YOUR SCRIPT FROM 03_problem_solution_script.md HERE]

--- PRODUCT ---
Product: [PRODUCT NAME]
Core Message: [1 sentence summary of the ad's key message]

--- YOUR TASK ---
Adapt the base script for each of the following platforms. For each platform, change:
1. Hook style (platform-specific opening behavior)
2. Pacing and rhythm (cuts per second, energy level)
3. Text overlay strategy (font style, frequency, word count)
4. CTA format (platform-native CTA language)
5. Caption format (length, hashtags, emojis)
6. Audio recommendation (trending sound vs. voiceover vs. music)

PLATFORM ADAPTATIONS NEEDED:

▶ TIKTOK
- Format: 9:16 vertical, 15–60 seconds
- Audience behavior: Fast scroll, sound-on culture, trends-first
- Hook style: First-person raw/authentic, jump cuts, trending audio
- CTA: "Comment [word] and I'll DM you the link" / "Link in bio"
- Caption: 2–3 sentences max + 3–5 hashtags

▶ INSTAGRAM REELS
- Format: 9:16 vertical, 15–90 seconds  
- Audience behavior: Aesthetics matter more, slightly longer attention
- Hook style: Visually polished but still authentic
- CTA: "Save this post" / "Link in bio" / "DM me [word]"
- Caption: Can be longer, storytelling works, up to 10 hashtags

▶ YOUTUBE SHORTS
- Format: 9:16 vertical, under 60 seconds
- Audience behavior: Discovery-driven, search intent
- Hook style: Clear value proposition upfront, educational tone works
- CTA: "Subscribe for more" / "Check the link below"
- Caption: SEO-optimized title, description with keywords

▶ FACEBOOK / META ADS
- Format: Feed (4:5 or 1:1) or Story (9:16)
- Audience behavior: Older demographic, sound-off by default
- Hook style: Text-heavy first frame, benefit-driven opening
- CTA: Button CTA ("Shop Now", "Learn More", "Get Offer")
- Caption: Professional tone, clear offer in first line

▶ PINTEREST VIDEO ADS
- Format: 2:3 vertical or 1:1 square, 6–15 seconds
- Audience behavior: Aspirational, discovery mindset, planning mode
- Hook style: Visual transformation, lifestyle-forward
- CTA: "Save for later" / "Try it now"
- Caption: Keyword-rich, aspirational language
```
