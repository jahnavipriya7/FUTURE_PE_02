# 📝 PROBLEM-SOLUTION SCRIPT GENERATOR

> **Full video script using the PAS framework (Problem → Agitate → Solution)**

---

## THE PROMPT

```
You are a UGC video script writer specializing in the PAS (Problem, Agitate, Solution) framework for direct-response social media ads.

Write a complete, production-ready UGC video script using the following details:

--- INPUT ---
Product Name: [PRODUCT NAME]
Product Description: [1–2 sentence description]
Target Audience: [WHO IS THIS FOR]
Primary Pain Point: [SPECIFIC PROBLEM THEY FACE]
Emotional Trigger: [e.g., frustration, embarrassment, fear of missing out, desire for status]
Key Result / Transformation: [WHAT THEY ACHIEVE WITH THE PRODUCT]
CTA: [CALL TO ACTION — e.g., "Link in bio for 20% off"]
Script Length: [15s / 30s / 45s / 60s]
Creator Persona: [e.g., everyday person, expert, mom, student, professional]
Platform: [TikTok / IG Reels / YouTube Shorts]

--- FRAMEWORK ---
Apply the PAS framework:

PROBLEM (20% of script)
- Open with a specific, relatable problem
- Use "I used to..." or "Have you ever..." language
- Show don't tell — describe a real scenario

AGITATE (30% of script)
- Deepen the pain — what does it cost them? (time, money, confidence, relationships)
- Use emotional amplifiers
- Reference failed alternatives they've tried

SOLUTION (35% of script)
- Introduce the product naturally, as if a friend is recommending it
- Focus on 2–3 benefits max, tied directly to the pain points
- Add social proof (numbers, before/after, testimonial language)

CTA (15% of script)
- Soft or hard CTA based on funnel stage
- Create urgency (limited time, limited stock, exclusive offer)
- Make the next step crystal clear

Format output as:
[TIMESTAMP] [VISUAL DIRECTION] [VOICEOVER / DIALOGUE] [TEXT OVERLAY]
```

---

## SCRIPT LENGTH GUIDE

| Length | Best For | Word Count |
|--------|----------|------------|
| 15s | Awareness, retargeting | 35–45 words |
| 30s | Main ad unit | 75–90 words |
| 45s | Storytelling, education | 110–130 words |
| 60s | Deep funnel, complex product | 150–175 words |
