# 🔀 MULTI-HOOK VARIATIONS — A/B Testing Generator

> **Generate 10+ hook variations for split testing. Find your winning creative.**

---

## THE PROMPT

```
You are a performance creative strategist specializing in paid social A/B testing.

I need to generate multiple hook variations for a UGC ad so I can split test them and find the highest-performing creative.

--- PRODUCT & AUDIENCE ---
Product: [PRODUCT NAME]
Core Benefit: [PRIMARY TRANSFORMATION/RESULT]
Target Audience: [WHO IS THIS FOR]
Pain Point: [CORE FRUSTRATION]
Platform: [TIKTOK / INSTAGRAM / YOUTUBE SHORTS]

--- YOUR TASK ---
Generate 10 hook variations across 5 different emotional angles:

ANGLE 1 — PAIN/FRUSTRATION (2 hooks)
Target the specific frustration the audience feels. Make them feel seen.

ANGLE 2 — CURIOSITY/INTRIGUE (2 hooks)  
Create an open loop they can't help but want to close. Tease without revealing.

ANGLE 3 — SOCIAL PROOF/FOMO (2 hooks)
Leverage community, numbers, or exclusivity. Make them feel behind.

ANGLE 4 — TRANSFORMATION/ASPIRATION (2 hooks)
Paint the "after" picture vividly. Sell the destination, not the vehicle.

ANGLE 5 — PATTERN INTERRUPT/SHOCK (2 hooks)
Say something unexpected, counterintuitive, or visually jarring.

For each hook variation, provide:
- The hook line (spoken + text overlay version)
- Recommended opening visual (what's on screen in the first 1.5s)
- Emotional trigger being activated
- Predicted audience that will respond best to this hook
- A/B test priority rating (1–5, where 5 = test first)
```

---

## A/B TESTING FRAMEWORK

### What to Test (Priority Order)

1. **Hook** — biggest impact on thumb stop rate
2. **Opening Visual** — face vs. product vs. text screen
3. **CTA** — soft vs. hard, urgency vs. value
4. **Script Length** — 15s vs. 30s vs. 45s
5. **Creator Persona** — different faces, demographics

### How to Structure Your Test

```
PHASE 1: Hook Test (Week 1–2)
- Keep everything else identical
- Test 3–5 hooks
- Measure: Thumb Stop Rate (>25% = good), 3-second view rate

PHASE 2: Creative Test (Week 3–4)
- Take winning hook
- Test different visual styles / creator types
- Measure: Watch-through rate, CTR

PHASE 3: CTA Test (Week 5–6)
- Take winning hook + creative
- Test 2–3 CTAs
- Measure: CTR, conversion rate, ROAS
```

### Key Metrics by Hook Type

| Hook Type | Primary KPI | Benchmark |
|-----------|------------|-----------|
| Pain/Frustration | Comment rate | >2% |
| Curiosity | Watch-through | >40% |
| Social Proof | CTR | >1.5% |
| Transformation | Save rate | >3% |
| Pattern Interrupt | Thumb stop | >30% |

---

## HOOK ROTATION STRATEGY

- **Week 1**: Run all hooks at equal budget, $5–$10/day each
- **Week 2**: Kill bottom 50%, double budget on top 2
- **Week 3**: Declare winner, create 3 variations of the winner
- **Ongoing**: Refresh hooks every 4–6 weeks to avoid creative fatigue
