## KEY DIFFERENCES: SAAS vs D2C

| Element | D2C (Physical) | SaaS (Digital) |
|---------|---------------|----------------|
| Hook angle | Emotional / appearance | Logical / time/money savings |
| Script tone | Relatable friend | Slightly more professional |
| Social proof | Reviews, before/after | User numbers, ratings, case studies |
| CTA | "Buy now, limited offer" | "Free trial, no risk" |
| Objection to address | "Does it actually work?" | "Is it worth paying for?" |
| Visual | Product in use | Screen recording + creator |
