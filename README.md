# FUTURE_PE_02 — AI-Powered UGC Ad & Content Marketing Prompt System

> **Turn any product into viral-ready UGC video scripts in minutes using AI.**

---

## 🚀 What Is This?

**FUTURE_PE_02** is a complete prompt engineering system designed for marketers, founders, and creators who want to generate high-converting **User-Generated Content (UGC) ads** for social media using AI (ChatGPT, Claude, Gemini, etc.).

This repository contains **8 battle-tested prompt modules**, ready-to-use templates, platform adaptation workflows, and real brand examples — everything you need to go from product idea to publish-ready video script in one session.

## ⚡ Quick Start

1. **Fill in your product details** → [`templates/product_input_template.md`](templates/product_input_template.md)
2. **Run the Master Prompt** → [`prompts/01_master_prompt.md`](prompts/01_master_prompt.md)
3. **Generate hooks** → [`prompts/02_hook_generator.md`](prompts/02_hook_generator.md)
4. **Choose your platform** → [`prompts/04_platform_adaptation.md`](prompts/04_platform_adaptation.md)
5. **Get your full content pack** → [`prompts/08_complete_content_pack.md`](prompts/08_complete_content_pack.md)
