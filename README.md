# FUTURE_PE_02 🚀
AI-Powered UGC Ad Prompt System for High-Converting Social Media Ads.

## What This Repository Does
This repository provides structured AI prompts to generate:
- Scroll-stopping hooks
- Conversion-focused UGC scripts
- Platform-specific adaptations
- Brand voice variations
- Psychological trigger integration
- Testing frameworks

## Supported Platforms
- Instagram Reels
- TikTok
- Facebook
- YouTube Shorts

## How To Use
1. Go to /templates/product_input_template.md
2. Fill in your product details
3. Use prompts inside /prompts
4. Generate scripts using AI
5. Adapt per platform
6. Test and iterate

## Outcome
Turn product details into ready-to-film UGC ads in minutes.
